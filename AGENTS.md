# Agent Workflow

## Full development cycle

When user requests "full cycle", "design then implement", or "design → implement → review → test", run these skills **in order**:

1. **architect-design** — Analyze requirements, propose design, document decisions
2. **developer-implement** — Implement according to Step 1 design, follow conventions
3. **architect-review** — Verify implementation matches design, flag deviations
4. **test-runner** — Add/run tests, fix failures

## How to trigger

- "Сделай фичу X по full cycle"
- "Design, implement, review, and test feature Y"
- "Полный цикл разработки для Z"
