# Безопасная установка OpenClaw / Clawdbot

## О чём инструкция

Речь идёт **не про удалённую работу из России как таковую**, а про **безопасный доступ к OpenClaw/Clawdbot**: как поднять сервис на VPS и открыть его только через приватную сеть, чтобы не светить его в интернет.

**Суть:** сервис крутится на VPS (например, DigitalOcean), а доступ к нему идёт только через **Tailscale** (приватная mesh-сеть). Порт сервиса закрыт фаерволом от публичного доступа — зайти можно только с устройств, подключённых к тому же Tailscale-аккаунту.

---

## Краткое содержание шагов

**Нужно:** VPS с OpenClaw в Docker, аккаунт Tailscale, SSH на VPS.

### Этап 1 — Частная сеть (Tailscale)

1. Установить Tailscale на VPS: `curl -fsSL tailscale.com/install | sh`
2. Включить и авторизоваться: `sudo tailscale up` → открыть ссылку в браузере, войти в аккаунт.
3. Проверить: `tailscale ip -4` → адрес вида `100.x.x.x`.
4. Узнать порт контейнера: `docker ps` (например, `44452`).
5. Отдать сервис только через Tailscale: `sudo tailscale serve --bg http://localhost:YOUR_PORT`.
6. Установить Tailscale на ноутбук/телефон и войти в тот же аккаунт — доступ к OpenClaw будет только с этих устройств.

### Этап 2 — Фаервол (UFW)

1. Разрешить SSH и трафик по Tailscale, закрыть порт сервиса снаружи:
   - `sudo ufw allow OpenSSH`
   - `sudo ufw allow in on tailscale0`
   - `sudo ufw deny YOUR_PORT`
2. Включить UFW: `sudo ufw enable` (если ещё не включён).

В итоге OpenClaw доступен только внутри Tailscale-сети, снаружи порт закрыт.

---

## Для чего это подключение

**Tailscale** здесь нужен, чтобы:

1. **Не светить сервис в интернет** — OpenClaw не виден по публичному IP и не сканируется ботами; снижается риск взлома и лишнего внимания.
2. **Доступ только «своим»** — зайти могут только устройства, которые вы добавили в свой Tailscale-аккаунт (ноутбук, телефон и т.д.).
3. **Удобно пользоваться откуда угодно** — не привязка к одному месту: включил Tailscale на устройстве — получил доступ к своему инстансу OpenClaw, как будто он в локальной сети.

То есть подключение через Tailscale — это **приватный, безопасный канал** к вашему боту на VPS вместо того, чтобы открывать его порт в интернет.
