# 844. Backspace String Compare — итоги разбора

Задача: две строки `s` и `t` вводятся в пустые редакторы; `'#'` — backspace (удаляет предыдущий символ). Вернуть `true`, если итоговые строки равны.

Решение: два указателя с конца строк; для каждой строки свой счётчик `skip`. При встрече `'#'` увеличиваем `skip` и сдвигаем указатель влево; при обычном символе: если `skip > 0`, уменьшаем `skip` и сдвигаем (символ «удалён»), иначе — нашли следующий символ результата, выходим из внутреннего цикла. Сравниваем пары символов; после совпадения делаем `i--`, `j--`. Выход из внешнего цикла по условию `i >= 0 && j >= 0`; если одна строка закончилась раньше, сравнение `s[i] !== t[j]` даёт `undefined !== char` и возвращается `false`.

---

## 1. Сложность

- **Время:** O(n + m), один проход с конца по каждой строке.
- **Память:** O(1), только указатели и счётчики `skip`.

---

## 2. Альтернативы

- **Сбор результата в строку/массив:** симулировать ввод слева направо (стек или строка), затем сравнить две итоговые строки. Время O(n + m), память O(n + m).
- **Один общий цикл с функцией «следующий символ»:** вынести поиск следующего значимого символа в хелпер — код чище, асимптотика та же.

---

## 3. Краевые случаи (покрыты тестами)

- Примеры из условия: оба становятся `"ac"`; оба `""`; одна строка `"c"`, другая `"b"`.
- Обе пустые; одна пустая, вторая после backspace тоже пустая (`"###"` и `"##"`).
- Нет backspace; разная длина результата (лишний символ в конце).
- Backspace в пустом редакторе — текст остаётся пустым (`"#"`, `"#a#"` → `""`).
- Много подряд backspace и один общий символ в конце.

---

## 4. Тест-кейсы (зафиксированы)

| Описание | s | t | Ожидание |
|----------|---|---|----------|
| Example 1 | "ab#c" | "ad#c" | true |
| Example 2 | "ab##" | "c#d#" | true |
| Example 3 | "a#c" | "b" | false |
| Обе пустые | "" | "" | true |
| Одна пустая, другая только backspace | "###" | "##" | true |
| Без backspace, одинаковые | "abc" | "abc" | true |
| Без backspace, разные | "abc" | "abd" | false |
| В s лишний символ | "ac" | "a" | false |
| В t лишний символ | "a" | "ac" | false |
| Только backspace | "#" | "#" | true |
| Backspace и один символ в конце | "#a#" | "#" | true |
| Много backspace, оба → "" | "aaa###" | "a#a#a#" | true |
| Много backspace, оба → "c" | "aa#a###c" | "c" | true |
